Para enviar un correo electrónico desde una página HTML, necesitarás usar un formulario de contacto y un lenguaje de programación del lado del servidor, como PHP.

Aquí te dejo un ejemplo básico de cómo podrías hacerlo:

  1. Crea un formulario en tu página HTML que incluya campos para el nombre, correo electrónico, asunto y mensaje del usuario. También incluye un botón de enviar.
html
<form action="send-email.php" method="post">
  <label for="name">Nombre:</label>
  <input type="text" id="name" name="name" required>
  
  <label for="email">Correo electrónico:</label>
  <input type="email" id="email" name="email" required>
  
  <label for="subject">Asunto:</label>
  <input type="text" id="subject" name="subject" required>
  
  <label for="message">Mensaje:</label>
  <textarea id="message" name="message" required></textarea>
  
  <input type="submit" value="Enviar">
</form>
  1. Crea un archivo PHP llamado "send-email.php" que manejará el envío del correo electrónico. En este archivo, utiliza la función "mail" de PHP para enviar el correo electrónico al destinatario deseado.
php
<?php
if(isset($_POST['email'])) {
 
    // Correo electrónico del destinatario
    $to = "destinatario@example.com";
 
    // Asunto del correo electrónico
    $subject = $_POST['subject'];
 
    // Contenido del correo electrónico
    $message = $_POST['message'] . "\n\nDe: " . $_POST['name'] . " <" . $_POST['email'] . ">";
 
    // Cabeceras del correo electrónico
    $headers = "De:" . $_POST['email'];
 
    // Envía el correo electrónico usando la función mail de PHP
    mail($to,$subject,$message,$headers);
 
    // Redirige al usuario a una página de confirmación
    header('Location: confirmation.html');
 
} else {
 
    // Si el usuario accede directamente a esta página sin enviar el formulario, redirige a la página de inicio
    header('Location: index.html');
 
}
?>

Este es solo un ejemplo básico, y es posible que necesites personalizarlo para adaptarlo a tus necesidades específicas. También ten en cuenta que el envío de correo electrónico desde PHP puede tener algunas limitaciones dependiendo del servidor de alojamiento que estés utilizando.

Un error HTTP 405 indica que el método de solicitud utilizado en tu formulario HTML no es compatible con el servidor web. En general, esto significa que estás intentando enviar el formulario usando un método que no está permitido, como GET en lugar de POST.

Para solucionar el error HTTP 405, asegúrate de que estás utilizando el método de solicitud correcto en tu formulario HTML. Por ejemplo, si tu archivo PHP de envío de correo electrónico utiliza el método POST, debes especificar method="post" en tu formulario HTML:

html
<form action="send-email.php" method="post">
  <!-- Aquí van los campos del formulario -->
  <input type="submit" value="Enviar">
</form>

Si aún recibes el error HTTP 405 después de verificar que estás utilizando el método de solicitud correcto, es posible que debas verificar la configuración del servidor web. Puedes consultar con tu proveedor de alojamiento web o con un administrador de sistemas para obtener más ayuda.

Sí, puede ser una buena opción usar una librería como PHPMailer para enviar correos electrónicos desde tu página web en lugar de usar la función mail() de PHP. PHPMailer es una biblioteca popular y bien documentada que proporciona una forma fácil y segura de enviar correos electrónicos desde PHP.

Para usar PHPMailer, primero debes descargar y descomprimir la biblioteca PHPMailer en tu servidor web. Luego, puedes crear un archivo PHP y agregar el siguiente código para configurar PHPMailer y enviar un correo electrónico:

php
<?php
// Incluye la clase PHPMailer
require_once('path/to/PHPMailer/src/PHPMailer.php');
require_once('path/to/PHPMailer/src/SMTP.php');

// Crea una nueva instancia de PHPMailer
$mail = new PHPMailer\PHPMailer\PHPMailer();

// Configura el servidor SMTP
$mail->isSMTP();
$mail->Host = 'smtp.gmail.com';
$mail->SMTPAuth = true;
$mail->Username = 'tu-correo@gmail.com';
$mail->Password = 'tu-contraseña';
$mail->SMTPSecure = 'tls';
$mail->Port = 587;

// Configura el correo electrónico
$mail->setFrom($_POST['correo'], $_POST['nombre']);
$mail->addAddress('destinatario@example.com');
$mail->Subject = 'Contacto desde tu sitio web';
$mail->Body = $_POST['mensaje'] . "\n\nDe: " . $_POST['nombre'] . " <" . $_POST['correo'] . ">" . " # " . $_POST['telefono'];

// Envía el correo electrónico
if ($mail->send()) {
    // Redirige al usuario a una página de confirmación
    header('Location: confirmation.html');
} else {
    echo 'Error al enviar el correo electrónico: ' . $mail->ErrorInfo;
}
?>

En este código, debes reemplazar path/to/PHPMailer con la ruta donde has descomprimido la biblioteca PHPMailer en tu servidor web, y tu-correo@gmail.com y tu-contraseña con tus credenciales de correo electrónico. También debes actualizar la dirección de correo electrónico del destinatario y cualquier otra información relevante.

Este código configura PHPMailer para usar el servidor SMTP de Gmail para enviar el correo electrónico. Si usas otro proveedor de correo electrónico, es posible que debas cambiar los valores de configuración para que coincidan con los de tu proveedor.
