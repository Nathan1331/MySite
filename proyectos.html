REACT Movie Land

MovieLand es un proyecto web desarrollado con tecnologías React, HTML y CSS, que permite a los usuarios buscar películas utilizando un API de películas. El objetivo principal del proyecto es proporcionar una herramienta de búsqueda de películas fácil de usar y accesible para los usuarios.
La interfaz de usuario de MovieLand es atractiva y funcional, lo que permite a los usuarios buscar películas utilizando el nombre de la película. Una vez que se introduce el nombre de la película, el sistema muestra una lista de títulos de películas y su género correspondiente, lo que permite a los usuarios encontrar fácilmente la película que están buscando.
El proyecto utiliza tecnologías de React para crear una interfaz de usuario rápida y sensible, que permite a los usuarios buscar películas sin demora. Además, se ha integrado un API de películas para garantizar que los resultados de búsqueda sean precisos y actualizados.

HTML/CSS Blog cafe

Blog Café es un proyecto web desarrollado con tecnologías HTML y CSS, que incluye una página de nosotros, cursos y contacto. El objetivo principal del proyecto es proporcionar a los usuarios un sitio web informativo y atractivo que promueva un negocio de cafetería y ofrezca información sobre los cursos que se imparten.
La página de nosotros presenta información detallada sobre el negocio de la cafetería, su historia y sus valores fundamentales. La página de cursos ofrece información sobre los cursos que se imparten, su duración y sus precios, así como un formulario de registro en línea para aquellos que deseen inscribirse.
La página de contacto incluye información de contacto detallada, así como un formulario de contacto en línea para que los clientes puedan enviar comentarios, preguntas y solicitudes directamente al equipo de la cafetería.

C# Nate Store

Esta tienda virtual es un proyecto desarrollado con tecnologías de ASP.NET C#, Entity Framework y SQL como base de datos. El proyecto permite a los usuarios crear productos y manipularlos a su gusto, incluyendo la opción de subir imágenes para ser almacenadas en la base de datos.
La interfaz de usuario es amigable y permite a los usuarios acceder a la funcionalidad de la tienda virtual. La creación de productos y la manipulación de datos se realiza a través de formularios en línea, que permiten a los usuarios agregar información detallada sobre cada producto y cargar imágenes para ser almacenadas en la base de datos.
El proyecto utiliza Entity Framework para administrar la conexión a la base de datos de SQL, lo que garantiza un acceso seguro y rápido a los datos. La inclusión de la funcionalidad de carga de imágenes y la manipulación de productos permite a los usuarios personalizar la tienda virtual según sus necesidades y preferencias.

C# Ruteo de Buses

C# Ruteo de Buses es un proyecto desarrollado con tecnologías de ASP.NET C#, Entity Framework y SQL como base de datos, junto con MongoDB para almacenar registros de bitácora de acciones. El proyecto tiene una funcionalidad de inicio de sesión para usuarios autorizados y permite realizar el ruteo de buses para un sistema de transporte.
El proyecto presenta una interfaz de usuario amigable que permite a los usuarios acceder a las funcionalidades de ruteo. Los datos del sistema de transporte se almacenan en la base de datos de SQL, mientras que los registros de bitácora se almacenan en la base de datos de MongoDB.
El proyecto también utiliza Entity Framework para administrar la conexión a la base de datos, Y se implementan los metodos de un CRUD (Create, Read, Update, Delete) utilizando el modelo de code first.

C# Nate Hotel

C# Nate Hotel es un proyecto desarrollado utilizando tecnologías de ASP.NET C#, Entity Framework y SQL como base de datos, junto con MongoDB para almacenar registros de bitácora de acciones. El objetivo principal del proyecto es permitir la creación de usuarios y reservaciones en las habitaciones de un hotel.
El proyecto presenta una interfaz de usuario intuitiva y fácil de usar, que permite a los usuarios registrarse en el sistema y realizar reservaciones en línea de forma rápida y eficiente. Los datos de reserva se almacenan en la base de datos de SQL, mientras que los registros de bitácora se almacenan en la base de datos de MongoDB. El proyecto también utiliza Entity Framework para administrar la conexión a la base de datos.
